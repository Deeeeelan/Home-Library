# Home Library
For OCSEFHacks-26

Home Library is a solution to help communities with no libraries (or libraries that are limited, low quality, or otherwise unusable) get access to books. It allows **any person** to put their own books for checking out by other users! Anyone can sign up, add some books, and search for other books near them.
